## Why No Compiled JAR?

The compiled, runnable JAR requires building with Gradle, which needs to download:

1. **ForgeGradle 2.2-SNAPSHOT** (~50 MB)
   - Minecraft mod build system
   - Handles deobfuscation and reobfuscation

2. **MixinGradle 0.6-SNAPSHOT** (~10 MB)
   - Bytecode transformation system
   - Applies Mixin injections

3. **Minecraft 1.8.9** (~10 MB)
   - Game files for compilation
   - Decompiled source code

4. **Forge 11.15.1.2318-1.8.9** (~4 MB)
   - Modding framework
   - Runtime libraries

5. **Workspace Setup** (~150 MB cached files)
   - Decompiled Minecraft source
   - Deobfuscation mappings

**Total download:** ~200-250 MB
**Total time (first build):** 10-30 minutes

## How to Get the Runnable JAR

### Method 1: Build on Your Machine (Recommended)

```bash
# Prerequisites: Java 8, Internet connection

# Clone the repository
git clone <repository-url>
cd RaspberryClient

# Run automated build
./build-client.sh

# Wait 10-30 minutes (first time only)
# Output: release/Raspberry-Client-1.0.0.jar
```

###  Method 2: Use GitHub Actions (CI/CD)

Set up GitHub Actions to automatically build on every commit. The workflow will:
1. Download all dependencies
2. Setup workspace
3. Compile the mod
4. Upload JAR as artifact

### Method 3: Manual Gradle Build

```bash
gradle wrapper --gradle-version 2.14
./gradlew setupDecompWorkspace  # 10-30 min
./gradlew build                  # 30-60 sec
```

## Installation (After Building)

Once you have the compiled JAR:

1. **Install Prerequisites:**
   - Minecraft 1.8.9
   - Minecraft Forge 1.8.9-11.15.1.2318

2. **Install Mod:**
   ```bash
   cp Raspberry-Client-1.0.0.jar ~/.minecraft/mods/
   ```

3. **Launch:**
   - Open Minecraft Launcher
   - Select "Forge 1.8.9" profile
   - Click "Play"

4. **Verify:**
   - HUD elements appear (FPS, CPS, etc.)
   - Press Right Shift to open ClickGUI
   - Check mods list shows "Raspberry Client v1.0.0"
